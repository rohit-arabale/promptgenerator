## How It Works

Rohit Arabale's promptgenerator uses a multi-step generation process to create highly effective AI prompts:

1. **Initial Enhancement**

   - Takes your input prompt
   - Uses GPT-4 to enhance and structure it professionally
   - Adds missing context and improves clarity

2. **Three-Step Refinement**

   - Step 1: Converts the enhanced prompt into a structured prompt format
   - Step 2: Adds stricter guidelines and specific examples
   - Step 3: Final optimization with additional safety measures and clear instructions

3. **Final Output**

   - Delivers a production-ready prompt
   - Includes proper formatting and structure
   - Ensures compatibility with various AI models

This iterative process ensures each generated prompt is:

- Well-structured
- Comprehensive
- Clear and concise
- Safe and controlled
- Ready for immediate use

## Getting Started

1. Clone Rohit Arabale's repository:

   ```sh
   git clone https://github.com/rohit-arabale/promptgenerator.git
   cd promptgenerator
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   pnpm install
   ```

3. Create a `.env.local` file in the root directory:

   ```bash
   OPENAI_API_KEY=your_openai_api_key_here
   PRE_PROMPT='Your pre-prompt here'
   SYSTEM_PROMPT='Your system prompt here'
   UPSTASH_REDIS_REST_URL="https://your-upstash-url.upstash.io"
   UPSTASH_REDIS_REST_TOKEN="your_upstash_token_here"
   ```

4. Run the development server:

   ```bash
   npm run dev
   # or
   pnpm dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see Rohit Arabale's promptgenerator.
